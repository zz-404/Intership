## CNN(convolutional neural network) 卷积神经网络 （图像类）
```
卷积层（提取特征）
池化层（降维）
HWC -> 高，宽，通道
32 * 32 *3
层在处理时 3 * 32 * 32
全连接层

图像分类 用cifar10 数据集
```

## RNN(recurrent）循环神经网络 （文本类）
```
词嵌入层
循环网络层
全连接层

歌词AI生成器

朴素贝叶斯：唯一一个靠概率就能做分类的机器学习算法
```
