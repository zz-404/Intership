## CNN(convolutional neural network) 卷积神经网络 （图像类）
```
卷积层（提取特征）
池化层（降维）
HWC -> 高，宽，通道
32 * 32 *3
层在处理时 3 * 32 * 32
全连接层

图像分类 用cifar10 数据集
```

## RNN(recurrent）循环神经网络 （文本类）
```
词嵌入层
循环网络层
全连接层

歌词AI生成器

朴素贝叶斯：唯一一个靠概率就能做分类的机器学习算法
```
## ANN (artificial） 人工神经网络（只能处理二维数据）
```
输入层（1）
隐藏层（n）
输出层（1）

深度学习的可解释性差
```

## 张量
```
Tensor

张量中只能是数值
一个神经元干了两件事，一个是加权求和，一个是激活函数，激活函数把你加权求和的结果映射成一个概率

如果没有激活函数则只能做线性问题，做不了分类问题

深度学习既能做分类，也能做回归

深度学习尤其擅长处理高维数据，如图像，语音，文本
机器学习依赖人工提取特征，并设计算法
深度学习依赖算法自动提取特征

语音 -> 频谱图
```
